# Version 1.5.0

**New parcel sizes**
- Added Narrow (1 cell wide) parcels
- Added Extra-Wide (7,8 cells wide) parcels 

**Snapping improvements**
- Revamped snapping and added snap support for start and end of line tool
- Refactored the way snapping injects itself into the base game
- Improved logic and tweaked parameters for various snapping options, especially when multiple options are used together.
- Improved the way platter let's the base game know a snap has occurred, which for example should remove the "rotation" options from view when snapped.

**Notable Bug Fixes**
- Fixed: Unzoned parcel cells would be "eaten" by the vanilla grid in some cases
- Fixed: Game would crash when drawing roads in some cases
- Fixed: Parcels would connect to unzoned RoadBuilder roads, causing issues
- Fixed: Relocate and Picker would not work correctly when used with Parcels 
- Fixed: Parcels would cause some buildings to lose props and trees when growing
- Fixed: Zone indexes would get out of sync when adding or removing zones between save loads, causing incorrect parcel colors and prezoning behavior

**Other notable improvements**
- Improved the building count tooltip accuracy and clarity
- Adds keyboard-based controls for parcel sizes

**Other improvements**
- Adjusted opacity of the parcel overlay
- Added a colored outline to unzoned Platter cells to distinguish them from vanilla unzoned cells
- Improved performance when caching and delivering data to the UI, especially for the pre-zone panel
- Improved the way the road connection icon is added and removed
- Improved performance of internal prefab selection mechanisms by caching references to important prefabs in memory
- Numerous performance improvements to core systems/queries
- Updated translations
- Fixed version mismatch between PDXMods version and binary version, which caused some confusion and issues with mod updates.
