# Version 1.6.0
- New parcel selection UI: 
	- The toolbar now shows a single "parcel" entry instead of 30 size variants
- Richer building per zone per lot information:
	- Parcel size selection can now be made via a new grid UI in the side panel, or as before via shortcuts or scrollwheel
	- Building counts are now surfaced in both directions: the sidepanel UI shows how many buildings are available for the selected pre-zone at each parcel size, and the Pre-Zone dropdownshows how many buildings are available for the currently selected parcel size in each zone.
- Translations have been refreshed across all supported languages, thanks to the many community volunteers on Crowdin
